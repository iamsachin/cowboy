    - Subscribe to WebSocket onDataChanged for live updates (same as useAnalytics).
    - `modelDistribution` is `ref<ModelDistributionEntry[]>([])`.
    - This is NOT a singleton composable -- each per-agent tab creates its own instance.

    - Export `useAgentComparison()` that returns `{ claudeCode, cursor, loading, error }`.
    - Internally creates two data streams -- one for 'claude-code' and one for 'cursor' -- by calling the API directly (not useAgentAnalytics, to avoid the watcher overhead).
    - `claudeCode` and `cursor` are each `{ overview: Ref<OverviewStats | null>, timeseries: Ref<TimeSeriesPoint[]>, modelDistribution: Ref<ModelDistributionEntry[]> }`.
    - Fetches both agents' data in parallel with `Promise.all`.
    - Watches `dateRange` for refetch. Subscribes to WebSocket onDataChanged.

    **4. Create AgentBadge.vue:**
    - Props: `agent: string`.
    - Renders a DaisyUI `badge badge-sm` with agent-specific color class.
    - Import `AGENT_LABELS` from `../utils/agent-constants`.
    - `badge-primary` for claude-code (sky blue theme), `badge-secondary` for cursor (purple theme).
    - Label from AGENT_LABELS lookup, fallback to raw agent string.

    **5. Create ModelDistributionChart.vue:**
    - Props: `data: ModelDistributionEntry[]`.
    - Uses vue-chartjs `Doughnut` component (import from 'vue-chartjs', register ArcElement, Tooltip, Legend from 'chart.js').
    - Labels = model names, data = totalTokens per model.
    - Use a palette of 6-8 distinct colors for segments.
    - Night theme compatible: transparent background, light text for labels.
    - Wrapped in `bg-base-200 rounded-lg p-4` with title "Model Distribution".
    - Show "No data" placeholder when data is empty.

    **6. Create ComparisonCard.vue:**
    - Props: `title: string`, `claudeValue: string`, `cursorValue: string`, `claudeTrend?: number | null`, `cursorTrend?: number | null`, `icon: Component`.

    - Uses DaisyUI card classes: `bg-base-200 rounded-lg p-4`.
    - Trend arrows if provided (same pattern as KpiCard).

    - Props: `claudeData: TimeSeriesPoint[]`, `cursorData: TimeSeriesPoint[]`, `metric: 'tokens' | 'cost' | 'conversations'`, `title: string`.

    - Dataset 1: Claude Code with AGENT_COLORS['claude-code'] colors.
    - Dataset 2: Cursor with AGENT_COLORS['cursor'] colors.
    - Metric determines which field to plot: 'tokens' = inputTokens+outputTokens, 'cost' = cost, 'conversations' = conversationCount.
    - Periods (x-axis labels) derived from union of both datasets' periods, sorted.
    - Night theme: transparent background, gridline color from CSS variable, light text.
    - Wrapped in `bg-base-200 rounded-lg p-4`.

    **8. Create AgentActivityChart.vue:**
    - Props: `claudeData: TimeSeriesPoint[]`, `cursorData: TimeSeriesPoint[]`.
    - Uses vue-chartjs `Bar` component with stacked bars.

    - Title: "Conversation Activity".
    - Agent colors applied per dataset. Night theme compatible.
    - Wrapped in `bg-base-200 rounded-lg p-4`.
