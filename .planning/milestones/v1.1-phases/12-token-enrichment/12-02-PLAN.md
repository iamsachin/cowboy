    4. **AssistantGroupCard.vue**: Accept new optional prop `tokenUsageByMessage?: Record<string, MessageTokenUsage>`. Add a `groupTokens` computed that iterates `props.group.turns`, looks up each `turn.message.id` in the map, and sums `inputTokens`, `outputTokens`, `cacheReadTokens`, `cacheCreationTokens`, and `cost`. Returns null if no data found. Display in the summary header after the duration span:
       ```html
       <span v-if="groupTokens" class="text-base-content/50">
         {{ formatTokenCount(groupTokens.inputTokens) }} in / {{ formatTokenCount(groupTokens.outputTokens) }} out
       </span>
       <span v-if="groupTokens?.cost != null" class="text-success/70">
         {{ formatTurnCost(groupTokens.cost) }}
       </span>
       ```
       Import `formatTokenCount` and `formatTurnCost` from `../utils/format-tokens`.

    5. **Per-turn tokens in expanded view**: AssistantGroupCard's expanded view renders individual turns inline (not via TurnCard component). Add token info per turn in the expanded section. After the turn's tool calls div, add:
       ```html
       <div v-if="getTurnTokens(turn)" class="text-xs text-base-content/40 mt-1">
         {{ formatTokenCount(getTurnTokens(turn)!.inputTokens) }} in / {{ formatTokenCount(getTurnTokens(turn)!.outputTokens) }} out
         <span v-if="getTurnTokens(turn)!.cost != null"> · {{ formatTurnCost(getTurnTokens(turn)!.cost!) }}</span>
       </div>
       ```
