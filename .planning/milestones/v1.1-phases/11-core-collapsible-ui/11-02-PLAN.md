<task type="auto">
  <name>Task 1: Refactor TurnCard with collapsed/expanded states and summary header</name>
  <files>packages/frontend/src/components/TurnCard.vue</files>
  <action>
    Rewrite TurnCard.vue to support two states controlled via props:

    **New props:**
    - `turn: AssistantTurn` (existing)
    - `expanded: boolean` (new -- controlled by parent)
    - `nextTurn?: Turn` (new -- for duration calculation)

    **New emits:**
    - `toggle` -- fired when user clicks the summary header

    **Collapsed state (when expanded=false):**
    Show a two-line summary header inside a `bg-base-200 border border-base-300 rounded-lg` container.

    Line 1 (metadata row): `flex items-center gap-2 text-xs`
    - ChevronDown icon (w-4 h-4, shrink-0) -- NOT rotated when collapsed
    - Model badge: `badge badge-sm badge-ghost` showing `turn.message.model` (if present)
    - Tool call count: "N tool call(s)" in `text-base-content/50` (if toolCalls.length > 0)
    - Duration: computed via `calculateDuration(turn, nextTurn)` in `text-base-content/50` (if not null)
    - Timestamp: `formatTime(turn.message.createdAt)` in `text-base-content/40 ml-auto`

    Line 2 (preview): `text-xs text-base-content/50 truncate pl-6`
    - Content from `getPreviewSnippet(turn)`

    The entire header div gets `@click="$emit('toggle')"` and `cursor-pointer select-none py-2 px-4`.

    **Expanded state (when expanded=true):**
    Same header but ChevronDown gets `rotate-180` class (CSS transform, the chevron points up).

    Below the header, render the full content (using v-if="expanded"):
    - Assistant text body (existing parsedContent rendering with CodeBlock)
    - Thinking section (existing details/summary with Brain icon -- unchanged)
    - Tool call list (existing ToolCallRow components)

    The expanded content section gets `px-4 pb-4 pt-2`.

    **Styling:** Keep the existing `bg-base-200 border border-base-300 rounded-lg` container but remove `max-w-[85%]` constraint (collapsed headers should span full width). Remove the old "Assistant" label and standalone timestamp from the header since the summary header replaces them.

    Import `ChevronDown` from lucide-vue-next. Import `getPreviewSnippet`, `calculateDuration` from `../utils/turn-helpers`. Keep existing imports for parseContent, formatTime, stripXmlTags, Brain, CodeBlock, ToolCallRowComponent.
  </action>
