Create three new Vue components:

**SystemMessageIndicator.vue** — Props: `{ group: SystemGroup }`
- Collapsed state (default): A thin, centered bar/chip styled as a timeline event, not a message bubble. Show count and type hints, e.g., "3 system messages (skill instruction, objective)". Use muted neutral styling (`bg-base-300/50 text-base-content/40 text-xs`). Include a small ChevronDown icon for expand affordance.
- Clicking toggles expanded state (local `ref`).
- Expanded state: Show each message in the group with:
  - A small category label badge (e.g., "Skill instruction", "System reminder", "Objective") using DaisyUI `badge badge-ghost badge-xs`.
  - The message content run through `stripXmlTags()` for readability, displayed in a muted regular font (`text-xs text-base-content/60`), max-height with overflow-y-auto to prevent walls of text. Use `max-h-40` with scroll.
  - Map category enum to human-readable labels: 'system-reminder' -> 'System reminder', 'skill-instruction' -> 'Skill instruction', 'objective' -> 'Objective', 'system-caveat' -> 'System caveat', 'other' -> 'System'
- The indicator should be centered in the message flow (not left or right aligned like user/assistant bubbles). Use `flex justify-center`.
- Use ChevronDown with rotate-180 transition pattern from AssistantGroupCard.vue for consistency.

**SlashCommandChip.vue** — Props: `{ turn: SlashCommandTurn }`
- Render as a compact pill-shaped badge, right-aligned like user messages (`flex justify-end`).
- Use DaisyUI badge styling: `badge badge-outline text-xs font-mono` with a subtle border.
- Display the command text (e.g., `/gsd:execute-phase 11`).
- Include a small Terminal icon from lucide-vue-next before the command text.
- Show timestamp in the same position as ChatMessage (small text above).

- Visually communicates "everything above is old context."
- Centered text on the divider line.
