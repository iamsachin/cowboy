<task type="auto">
  <name>Task 1: Wire collapse state and sticky toolbar into ConversationDetail</name>
  <files>packages/frontend/src/components/ConversationDetail.vue</files>
  <action>
    Rewrite ConversationDetail.vue to:

    1. **Instantiate useCollapseState composable:**
       ```typescript
       const { isExpanded, toggle, expandAll, collapseAll, expandedCount } = useCollapseState();
       ```

    2. **Compute assistant turn IDs** for expand-all:
       ```typescript
       const assistantTurnIds = computed(() =>
         turns.value.filter(t => t.type === 'assistant').map(t => t.message.id)
       );
       const totalAssistantTurns = computed(() => assistantTurnIds.value.length);
       const allExpanded = computed(() => expandedCount.value === totalAssistantTurns.value && totalAssistantTurns.value > 0);
       ```

    3. **Toggle all handler:**
       ```typescript
       function toggleAll() {
         if (allExpanded.value) {
           collapseAll();
         } else {
           expandAll(assistantTurnIds.value);
         }
       }
       ```

    4. **Sticky toolbar** above the turn list:
       ```html
       <div class="sticky top-0 z-10 bg-base-200/95 backdrop-blur-sm border-b border-base-300 px-4 py-2 flex items-center justify-between rounded-t-lg mb-2">
         <span class="text-xs text-base-content/60">
           {{ expandedCount > 0 ? `${expandedCount} of ${totalAssistantTurns} expanded` : `${totalAssistantTurns} turns` }}
         </span>
         <div class="tooltip tooltip-left" :data-tip="allExpanded ? 'Collapse All' : 'Expand All'">
           <button class="btn btn-ghost btn-xs" @click="toggleAll">
             <ChevronsUp v-if="allExpanded" class="w-4 h-4" />
             <ChevronsDown v-else class="w-4 h-4" />
           </button>
         </div>
       </div>
       ```

    5. **Pass props to TurnCard:**
       Update the template loop to pass `expanded`, `nextTurn`, and handle `toggle` emit:
       ```html
       <template v-for="(turn, idx) in turns" :key="turn.message.id">
         <ChatMessage v-if="turn.type === 'user'" :message="turn.message" />
         <TurnCard
           v-else
           :turn="turn"
           :expanded="isExpanded(turn.message.id)"
           :nextTurn="turns[idx + 1]"
           @toggle="toggle(turn.message.id)"
         />
       </template>
       ```

    6. **Imports:**
       Add `import { useCollapseState } from '../composables/useCollapseState';`
       Add `import { ChevronsDown, ChevronsUp } from 'lucide-vue-next';`
       Keep existing imports for groupTurns, ChatMessage, TurnCard.

    The composable is instantiated inside ConversationDetail.vue's setup function, so it naturally resets when the component re-renders for a new conversation (new messages prop = new component instance via Vue's key mechanism in the parent).
