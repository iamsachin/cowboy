  Add progressive rendering to ConversationDetail.vue using a simple "load more" pattern:

  1. Add constants and reactive state in the script section:
  ```typescript
  const PAGE_SIZE = 50;
  const visibleCount = ref(PAGE_SIZE);
  ```

  2. Add computed properties for visible turns:
  ```typescript
  const visibleTurns = computed(() => turns.value.slice(0, visibleCount.value));
  const hasMore = computed(() => visibleCount.value < turns.value.length);
  const remainingCount = computed(() => turns.value.length - visibleCount.value);
  ```

  3. Add a loadMore function:
  ```typescript
  function loadMore() {
    visibleCount.value = Math.min(visibleCount.value + PAGE_SIZE, turns.value.length);
  }
  ```

  4. Reset visibleCount when the message list changes (new conversation loaded):
  ```typescript
  watch(() => props.messages.length, () => {
    visibleCount.value = PAGE_SIZE;
  });
  ```
  Import `watch` and `ref` from vue (ref may already be imported).

  5. In the template, replace `turns` with `visibleTurns` in the v-for loop:
  ```vue
  <template v-for="turn in visibleTurns" :key="turnKey(turn)">
  ```

  6. Add a "Load more" button after the turn list template, before the closing `</div>`:
  ```vue
  <button
    v-if="hasMore"
    class="btn btn-ghost btn-sm w-full mt-2"
    @click="loadMore"
  >
    Load more ({{ remainingCount }} remaining)
  </button>
  ```

  7. Update `groupIds` computed to use `visibleTurns` instead of `turns` for the expand/collapse all toolbar:
