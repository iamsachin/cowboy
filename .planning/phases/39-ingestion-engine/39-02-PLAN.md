<objective>
Port the plan extraction engine, subagent linking algorithm, and subagent summarizer from Node.js to Rust.

Purpose: Plans and subagent links are higher-order features built on top of the parsed conversation data. Plan extraction uses heuristic pattern matching to identify multi-step plans in assistant messages. Subagent linking connects child conversations to their parent tool calls using a three-phase matching algorithm.
Output: Plan extractor, subagent linker, and subagent summarizer modules ready for orchestrator integration.
</objective>
