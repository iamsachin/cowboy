    - User events get label = first 30 chars of content + ellipsis if truncated
    - Assistant-group events get label = "model + N tools" format (e.g., "Opus 4 · 3 tools"), no tools suffix when 0
    - Compaction events get label = "-Nk tokens" format (e.g., "-12k tokens"), fallback "Compaction" when no delta
    - System groups, slash commands, clear dividers, agent prompts are excluded
    - turnIndex matches the index in the input array (for pagination boundary checks)
    - Panel state: isOpen defaults to true when no localStorage key exists
    - Panel state: isOpen reads from localStorage key 'timeline-panel-open'
    - Panel state: toggle() flips isOpen and writes to localStorage
    - Panel state: isOpen is false when localStorage has 'false'
  </behavior>
  <action>
    Create `packages/frontend/src/composables/useTimeline.ts` as a singleton composable (module-level refs, following useTokenRate pattern):

    1. **TimelineEvent interface:**
       ```typescript
       export interface TimelineEvent {
         key: string;           // turnKey for the grouped turn
         type: 'user' | 'assistant-group' | 'compaction';
         label: string;         // Truncated display text
         turnIndex: number;     // Index in full turns array
       }
       ```

    2. **extractTimelineEvents(turns: GroupedTurn[]): TimelineEvent[]** -- Pure function (exported separately for testing). Iterate turns with index, filter to 3 types:
       - `user`: key = turn.message.id, label = content.slice(0, 30) + ellipsis if longer
       - `assistant-group`: key = turn.turns[0].message.id, label = `${model || 'Assistant'}` + ` · N tool(s)` when toolCallCount > 0
       - `compaction`: key = turn.id, label = `-${Math.round(delta/1000)}k tokens` when delta > 0, else 'Compaction'
       - Skip all other types (system-group, slash-command, clear-divider, agent-prompt)

    3. **useTimeline() composable:**
       - Module-level `isOpen` ref initialized from localStorage key 'timeline-panel-open' (default true -- `!== 'false'`)
       - `toggle()` flips isOpen and persists to localStorage
       - `activeKey` ref<string | null> for currently highlighted event (set externally by scroll sync in Plan 02)
       - `setActiveKey(key: string | null)` setter
       - Return: `{ isOpen, toggle, activeKey, setActiveKey }`

    Write tests FIRST in `packages/frontend/tests/composables/useTimeline.test.ts`:
    - Mock localStorage with vi.stubGlobal
    - Test extractTimelineEvents with a mixed array of all 7 GroupedTurn types, verify only 3 types returned
    - Test label formatting for each type (truncation, model+tools, token delta)
    - Test panel defaults open when no localStorage
    - Test panel reads 'false' from localStorage correctly
    - Test toggle writes to localStorage
